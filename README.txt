본 패치는 macOS(Apple Silicon 포함) 최신 버전을 기준으로 제작되었습니다. 윈도우 버전에서는 resources.assets 파일 구조가 달라 호환되지 않을 수 있습니다.

자막 파일만 교체하는 방식이 아니라, 폰트 데이터를 안전하게 이식하여 
ㅁㅁㅁ 깨짐 현상을 방지했습니다.

-----------------------------------------------------------
[구성 파일]
1. resources.assets : 한글 폰트가 이식된 핵심 데이터 파일
2. en.zip : 한글화된 자막 파일 묶음
-----------------------------------------------------------

[설치 방법]

1. 게임 데이터 경로 찾기
   - 스팀 라이브러리 -> Dead Reset 우클릭 -> 관리 -> 로컬 파일 탐색
   - (경로 예시: DeadReset.app/Contents/Resources/Data/)

2. 폰트 데이터 적용
   - 다운로드한 [resources.assets] 파일을 위 경로에 덮어씌웁니다.
   - ※ 주의: 만약의 상황을 대비해 원본 resources.assets 파일은 백업해두세요.

3. 자막 파일 적용
   - 게임 경로 내 [Dead Reset/Subtitles/] 폴더로 이동합니다.
   - 기존의 [en.zip] 파일을 삭제하거나 백업한 후, 다운로드한 [en.zip]을 넣습니다.

4. 게임 실행 및 설정
   - 게임을 실행한 후 [Options] -> [Language]를 "English"로 설정합니다.
   - (영어 설정에서 한글 자막이 출력되도록 설계되었습니다.)

-----------------------------------------------------------
[참고 사항]
- 인터페이스(메뉴)는 안정성을 위해 영문으로 유지되며, 자막만 한글로 출력됩니다.
- 중국어(zhs) 설정 시 글자가 깨질 수 있으니 반드시 "English" 설정을 유지해 주세요.
